# 安装与配置

本节介绍 THETA 的安装步骤和环境配置方法。

## 安装步骤
1. 克隆仓库：
   ```bash
   git clone https://github.com/CodeSoul-co/THETA.git
   ```
2. 进入项目目录：
   ```bash
   cd THETA
   ```
3. 安装依赖：
   ```bash
   # 以 Python 项目为例
   pip install -r requirements.txt
   ```

## 环境配置
- 支持多平台部署（Linux/macOS/Windows）
- 推荐使用虚拟环境

更多详细配置请参考项目根目录下的 README 文件。
